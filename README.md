# Crack Segmentation WPF Application

- A .NET desktop application for crack detection and segmentation using the OmniCrack30k model converted to ONNX format.
- Original Repo https://github.com/ben-z-original/omnicrack30k
- Model converted to ONNX format using nnU-Net framework and ONNX Runtime for inference.
- Models are on GDrive
